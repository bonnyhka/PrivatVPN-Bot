# PrivatVPN: Инфраструктура и Бот (Глобальное обновление)

Современная VPN-платформа, ориентированная на обход жестких блокировок и высокую производительность. Проект объединяет веб-интерфейс на Next.js, Telegram-бота и продвинутую систему роутинга на базе sing-box.

## 🚀 Глобальные изменения (2026)
Проект прошел полную инвентаризацию и техническую очистку:
*   **Актуализация протоколов**: Удалены неиспользуемые протоколы (Hysteria2), фокус смещен на максимально незаметный VLESS+Reality.
*   **Интеллектуальный туннель**: Реализована схема межсерверного Shadowsocks-туннелирования для обхода блокировок в РФ.
*   **ByeDPI Integration**: Внедрена технология `ciadpi` для локальной разблокировки YouTube без потери скорости.
*   **Реорганизация кода**: Все служебные скрипты структурированы по категориям в папке `scripts/`.

## 🛠 Технический стек и протоколы

### 1. VLESS + Reality (Основной)
Используется как основной метод подключения для всех клиентов. 
- **Маскировка**: Работает под видом трафика `dl.google.com` или `www.apple.com`.
- **Защита**: XTLS Vision включен для предотвращения активного сканирования.

### 2. Shadowsocks (Межсерверный туннель)
Используется исключительно для внутренней пересылки заблокированного трафика между российскими и европейскими узлами. Это обеспечивает доступ к Instagram/ChatGPT с сохранением российского IP для остальных сервисов.

### 3. ByeDPI
Локальный SOCKS-прокси на российских узлах, позволяющий смотреть YouTube в 4K без замедлений, не покидая сеть РФ.

## 🏗 Архитектура

- **Web App**: Next.js 15, Prisma ORM, админ-панель для управления пользователями и нодами.
- **Telegram Bot**: Полный цикл взаимодействия с пользователем (покупка, выбор локаций, поддержка).
- **Automation**: Автоматизированные скрипты в `scripts/core` для синхронизации статистики и мониторинга здоровья серверов.

## 📁 Структура скриптов
- `scripts/core/` — Жизненно важные процессы (синхронизация трафика, применение фиксов).
- `scripts/diagnostics/` — Инструменты для глубокой проверки связи и отладки.
- `scripts/maintenance/` — Обслуживание базы данных и лимитов.
